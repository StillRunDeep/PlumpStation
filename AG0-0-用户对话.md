# AG0-0 用户对话 Agent

## 一、职责概述

AG0-0 是雨水泵房智能设计系统的**入口 Agent**，负责通过自然语言对话的方式引导用户逐步提供设计所需的核心参数，包括：

- **Q**：泵站总设计流量（m³/h）
- **N**：工作水泵台数（台）
- **S**：集水池占地面积（m²）

收集完毕后，AG0-0 将参数打包，分别传递给下游计算 Agent（AG1-1、AG1-2、AG2-1）。

---

## 二、在系统中的位置

```
用户
  │
  ▼
AG0-0（用户对话）
  │  输出：Q, N, S
  ├──► AG1-1（单泵规格计算）
  ├──► AG1-2（水泵维护间尺寸计算）
  └──► AG2-1（水池体积深度计算）
```

---

## 三、输入

| 参数 | 来源 | 说明 |
|------|------|------|
| 用户自然语言输入 | 对话界面 | 用户对项目背景的文字描述 |

---

## 四、输出

| 参数名 | 类型 | 单位 | 说明 |
|--------|------|------|------|
| Q | Float | m³/h | 泵站总设计流量，对应 `RainfallPumpStation.designFlowRate` |
| N | Int | 台 | 工作水泵台数（不含备用泵），对应 `Pump` 对象数量 |
| S | Float | m² | 集水池占地面积，用于推算池深，对应 `CollectionPool` 平面尺寸 |

---

## 五、对话流程

### 步骤 1：获取总设计流量 Q

Agent 提问：

> 请问本项目的泵站总设计排水流量 Q 是多少？（单位：m³/h 或 L/s）
>
> 如暂无明确数值，可告知汇水面积 A（m²）、径流系数 ψ 及设计降雨强度 i，由系统辅助估算。

**辅助计算（当用户提供汇水面积法参数时）：**

依据本体规则 **R-HY-05（设计流量计算规则）**：

```
Q = ψ × i × A
```

- ψ：径流系数，对应 `RainfallPumpStation.runoffCoefficient`，范围 0 < ψ ≤ 1
- i：设计降雨强度（m³/(s·m²)）
- A：汇水面积（m²），对应 `RainfallPumpStation.catchmentArea`

**校验规则：** Q > 0，否则请求重新输入。

---

### 步骤 2：获取工作水泵台数 N

Agent 提问：

> 请问本泵站计划安装几台工作水泵？（一般为 2～6 台，不含备用泵）

**说明：**
- 依据规则 **R-OP-02（并联泵台数限制规则）**：工作泵台数不应超过 6 台。
- 依据规则 **R-OP-03（备用泵配置规则）**：建议大型泵站额外配置 1 台备用泵（N+1 原则）。备用泵数量不计入 N。

**校验规则：**
- N 为正整数，且 2 ≤ N ≤ 6
- 如用户输入 N = 1，提示：单泵运行可靠性较低，建议至少 2 台

---

### 步骤 3：获取集水池占地面积 S

Agent 提问：

> 请问集水池的平面占地面积 S 大约是多少？（单位：m²）
>
> 若暂无确切数值，可提供集水池长度和宽度（m），由系统计算面积。

**说明：**
- S 是集水池（`CollectionPool`）平面尺寸约束，用于推算最小有效水深。
- 集水池底部须保持排泥坡度，依据规则 **R-CM-05（集水池底部坡度规则）**：坡度 ≥ 1:10。

**校验规则：** S > 0

---

### 步骤 4：确认与汇总

Agent 汇总已收集参数，请用户确认：

> 请确认以下设计参数：
> - 总设计流量 Q = \_\_\_ m³/h
> - 工作水泵台数 N = \_\_\_ 台
> - 集水池占地面积 S = \_\_\_ m²
>
> 参数是否正确？如需修改，请指出需要调整的项目。

---

## 六、参数校验规则汇总

| 参数 | 合法性约束 | 引用规则 |
|------|-----------|----------|
| Q | Q > 0，单位统一为 m³/h | R-HY-05 |
| N | 正整数，2 ≤ N ≤ 6 | R-OP-02 |
| S | S > 0，单位为 m² | R-CM-05 |

---

## 七、引用的本体对象与规则

**对象（Object）：**
- `RainfallPumpStation`：总设计流量（designFlowRate）、汇水面积（catchmentArea）、径流系数（runoffCoefficient）
- `Pump`：isStandby 属性区分工作泵与备用泵
- `CollectionPool`：集水池占地面积

**规则（Rule）：**
- **R-HY-05** 设计流量计算规则：Q = ψ × i × A
- **R-OP-02** 并联泵台数限制规则：工作泵 ≤ 6 台
- **R-OP-03** 备用泵配置规则（N+1）：建议配备备用泵
- **R-CM-05** 集水池底部坡度规则：坡度 ≥ 1:10

---

## 八、传递给下游 Agent 的参数包

```json
{
  "Q": <总设计流量，m³/h>,
  "N": <工作水泵台数，台>,
  "S": <集水池占地面积，m²>
}
```

下游 Agent 接收方：
- **AG1-1**：接收 Q、N → 计算单泵流量与排水管径
- **AG1-2**：接收 N → 计算水泵维护间尺寸
- **AG2-1**：接收 Q、N、S → 计算水池体积与水泵扬程
