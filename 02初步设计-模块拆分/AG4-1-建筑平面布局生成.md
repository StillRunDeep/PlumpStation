# AG4-1 建筑平面布局生成

> 模块版本：v2.0（含空间临近关系与交通组织）

---

## 一、模块目标

在 AG1-2（泵房尺寸）和 AG2-1（水池深度）结果确定后，自动生成 2～3 种完整建筑平面布局方案，供用户选择。每种方案须满足：

- 所有功能空间正确放置在对应楼层
- 有外部直接进入需求的房间必须临近外墙
- 起重机覆盖范围约束满足
- **功能关联房间通过走廊或大厅相互连通**

---

## 二、空间临近关系图（Adjacency Graph）

### 2.1 必须临近（MUST ADJACENT）

违反此类关系时，方案标记为"有约束未满足"，并扣分。

| 编号 | 房间 A | 房间 B | 原因 |
|------|--------|--------|------|
| M-01 | 总水表房 `meter_main` | 水表房 `meter_sub` | 两间水表房并排，共用同一外墙区段，便于计量管理 |
| M-02 | 水表房 `meter_sub` | 消防设备房 `fire_equip` | 小型服务用房聚集布置，减少外墙开门分散 |
| M-03 | 风机房 `fan_room` | 送货口2 `dock2` | 设备经吊装口直接进出风机房 |
| M-04 | 清洁泵房 `clean_pump` | 雨水回收房 `rainwater` | 两房共用管道竖井，设备关联度高 |

### 2.2 建议临近（SHOULD ADJACENT）

满足则加分，不满足不影响方案可行性。

| 编号 | 房间 A | 房间 B | 原因 |
|------|--------|--------|------|
| S-01 | 变压器房1 `trafo1` | 变压器房2 `trafo2` | 并排布置方便母线桥架连接，减少线路损耗 |
| S-02 | 低压配电室 `lv_control` | 一层走廊 `corridor_l1` | 控制室须通过内走廊进入，避免直接对外开门 |
| S-03 | 清洁泵房 `clean_pump` | 一层走廊 `corridor_l1` | 通过走廊进出，兼做管道检修通道 |
| S-04 | 雨水回收房 `rainwater` | 一层走廊 `corridor_l1` | 同上 |

## 三、交通组织（走廊/大厅）

### 3.1 地面层

| 空间 | 说明 |
|------|------|
| 维修大厅（停车+维修区） | 核心交通空间，15t 桥式起重机覆盖全区。设备经由卡车停泊→送货口1→泵室竖向进出。 |
| 小型服务房（水表、消防） | 沿外墙排列，各自独立对外门，无需内廊。 |
| 变压器房 | 直接对外设检修门，无内部流通需求。 |

> 地面层各房间均有外门，**不需要专用走廊**。维修大厅本身即为主要交通空间。

### 3.2 一层

一层设 **主走廊 `corridor_l1`**（最小净宽 1500 mm），连通以下房间：

```
[低压配电室] ←→ [走廊] ←→ [清洁泵房]
                  ↕
             [雨水回收房]
```

走廊位置：位于低压配电室与清洁泵/雨水回收房之间，同时与上方风机房区域分隔。

| 参数 | 数值 |
|------|------|
| 最小净宽 | 1500 mm |
| 典型净高 | ≥ 2400 mm |
| 典型位置 | 低压配电室西/东侧，纵向贯通一层南半部 |

## 四、三种模板描述

### 模板 A：变压器东置

```
地面层（Ground Floor）
┌─────────────────────────────────────┐
│ 总水表房 │ 水表房 │ 消防房 │ [trafo1] │
│ 停车区 + 维修区 (15t 桥吊)  │ [trafo2] │
└─────────────────────────────────────┘

一层（Level 1）
┌─────────────────────────────────────┐
│   风机房 (13600×11800, 5t吊)        │
├──────────┬───────────────────────────┤
│ 清洁泵房 │  [走廊]  │  低压配电室    │
│ 雨水回收 │          │  (8000×17500)  │
└──────────┴───────────────────────────┘
```

- 变压器集中在东侧，双面自然通风
- 走廊位于清洁泵/雨水房与低压配电室之间

### 模板 B：变压器西置

- 模板 A 东西镜像
- 适合场地西侧有变电站接入点的情况

### 模板 C：变压器北排

- 两台变压器并排于北侧，形成较宽较浅的体形
- 适合场地进深受限情况
- 一层走廊布局同模板 A

## 五、约束规则表

| 规则 ID | 类型 | 描述 | 违反后果 |
|---------|------|------|----------|
| C-01 | 必须 | 变压器/水表/消防房须有外墙接触（非南墙）| violation |
| C-02 | 必须 | 停车区、送货口1须在15t桥吊覆盖范围内 | violation |
| C-03 | 必须 | 风机房、送货口2须在5t单轨吊覆盖范围内 | violation |

| C-05 | 必须 | meter_sub ↔ fire_equip 必须临近 | violation |
| C-06 | 必须 | fan_room ↔ dock2 必须临近 | violation |

| C-08 | 建议 | trafo1 ↔ trafo2 建议临近 | 不加分 |

## 六、评分项说明

| 项目 | 分值 | 说明 |
|------|------|------|
| 基础分 | 10000 | — |
| 占地面积 | −8 pt/m² | 紧凑方案得分更高 |
| 必须临近满足 | +40/对 | 每满足一对 MUST 关系 |
| 建议临近满足 | +15/对 | 每满足一对 SHOULD 关系 |
| 走廊交通完整 | +20 | 走廊同时与 ≥2 个目标房间临近 |
| 变压器外墙 | +20/台 | 每台变压器接触东/西外墙 |
| 变压器同侧 | +20 | 两台变压器同侧布置 |
| 风机房近送货口 | 最高+30 | 距离线性衰减 |
| 约束违反 | −50/项 | 每项必须约束未满足 |

## 七、输入输出定义

**输入**：无（固定布局模板，与泵数量/尺寸无关）

**输出**：

```ts
interface LayoutVariant {
  template: Template       // 模板定义（A/B/C）
  placements: Placements  // 所有房间坐标 (mm)
  groundPlacements: Placements
  level1Placements: Placements
  violations: Violation[] // 约束违反列表
  adjacency: {
    satisfied: AdjPair[]  // 已满足的临近对
    violated: AdjPair[]   // 未满足的临近对
  }
  feasible: boolean       // violations.length === 0
  score: number           // 综合评分
}
```
