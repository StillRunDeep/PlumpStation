# AG1-2 水泵维护间尺寸计算 Agent

## 一、职责概述

AG1-2 负责根据工作水泵台数 N 及水泵外形尺寸，按照**水泵一字排开**的布置方式，计算水泵维护间（机房）的最小平面净尺寸，包括维护间净长 L 和净宽 W。计算结果遵循本体布置规则，确保相邻水泵间距和两侧距墙距离均满足检修要求。

---

## 二、在系统中的位置

```
AG0-0（用户对话）
  │  输入：N（台数）
  ▼
AG1-2（水泵维护间尺寸计算）
  │  输出：L（维护间净长）, W（维护间净宽）
  ▼
泵房平面布置设计
```

---

## 三、输入参数

| 参数名 | 类型 | 单位 | 来源 | 说明 |
|--------|------|------|------|------|
| N | Int | 台 | AG0-0 | 工作水泵台数（含备用泵时，取总台数） |
| w_pump | Float | m | 用户提供 / 默认值 | 单台水泵机组沿排列方向的外形宽度（含电机联轴器） |
| d_pump | Float | m | 用户提供 / 默认值 | 单台水泵机组沿通道方向的外形深度 |
| motorPower | Float | kW | 用户提供 / 可选 | 水泵配套电机功率，用于判断是否适用大型泵加大间距规则 |

**默认值（当用户未指定外形尺寸时）：**
- w_pump = 0.6 m（中小型离心泵典型值）
- d_pump = 0.8 m（含电机和联轴器的典型深度）

---

## 四、输出参数

| 参数名 | 类型 | 单位 | 说明 |
|--------|------|------|------|
| L | Float | m | 维护间净长（沿水泵排列方向） |
| W | Float | m | 维护间净宽（垂直于水泵排列方向，即维护通道方向） |
| d_spacing | Float | m | 相邻水泵净距（实际采用值） |
| e_wall | Float | m | 端部水泵距两侧墙净距（实际采用值） |

---

## 五、计算逻辑

### 步骤 1：确定泵间净距 d_spacing

依据规则 **R-LA-01（水泵间距规则）** 和 **R-LA-03（大型泵加大间距规则）**：

```
IF motorPower ≤ 55 kW:
    d_spacing = 1.0 m    （一般水泵，取下限）

IF motorPower > 55 kW:
    d_spacing = 1.2 m    （大型水泵，满足吊装检修要求）
```

> 在布置空间允许时，推荐取 d_spacing = 1.2 m，预留更充裕的检修空间。

---

### 步骤 2：确定端部距墙净距 e_wall

依据规则 **R-LA-02（泵与墙壁间距规则）** 和 **R-LA-03（大型泵加大间距规则）**：

```
IF motorPower ≤ 55 kW:
    e_wall = 0.8 m    （一般水泵）

IF motorPower > 55 kW:
    e_wall = 1.0 m    （大型水泵）
```

---

### 步骤 3：计算维护间净长 L

水泵一字排开（单排布置），沿排列方向：

```
L = N × w_pump + (N - 1) × d_spacing + 2 × e_wall
```

**分项说明：**
- `N × w_pump`：N 台水泵机组总宽度
- `(N - 1) × d_spacing`：相邻水泵之间共 (N-1) 个间距
- `2 × e_wall`：左右两端各留一个距墙距离

**向上取模：** 结果向上取至 0.1 m 精度。

**示例：** N = 3，w_pump = 0.6 m，d_spacing = 1.0 m，e_wall = 0.8 m
```
L = 3 × 0.6 + 2 × 1.0 + 2 × 0.8 = 1.8 + 2.0 + 1.6 = 5.4 m
```

---

### 步骤 4：计算维护间净宽 W

维护间净宽由以下三部分确定（取最大值）：

**（a）维护通道最小宽度**

依据规则 **R-LA-05（进水流速规则）** 中的通道要求，泵前维护通道净宽：

```
W_aisle ≥ 1.2 m    （最小维护通道净宽）
```

**（b）水泵机组深度加出水管空间**

```
W_equip = d_pump + 出水管及附件横向空间（约 0.5 m）
         = d_pump + 0.5 m
```

**（c）吊装设备轨道要求**

若机房设置电动葫芦等 `HoistingEquipment`，净宽须满足起重轨道跨度。

**综合取值：**

```
W = max(W_aisle, W_equip) + 0.3 m（余量）
```

向上取至 0.1 m 精度，且 **W ≥ 2.5 m**（工程经验最小值）。

**示例：** d_pump = 0.8 m
```
W_equip = 0.8 + 0.5 = 1.3 m
W = max(1.2, 1.3) + 0.3 = 1.6 m → 取 W = 2.5 m（不小于最小值）
```

---

### 步骤 5：布置合规性校验

对计算结果进行规则验证：

| 校验项 | 规则 | 约束 |
|--------|------|------|
| 相邻水泵净距 | R-LA-01 | ≥ 1.0 m（一般），≥ 1.2 m（大型泵） |
| 端部距墙净距 | R-LA-02 | ≥ 0.8 m（一般），≥ 1.0 m（大型泵） |
| 工作泵台数 | R-OP-02 | N ≤ 6 台 |
| 同型号水泵 | R-OP-01 | 建议同型号，w_pump 相同 |
| 布置对称性 | R-OP-05 | 建议左右对称布置 |

## 七、引用的本体对象与规则

**对象（Object）：**
- `Pump`：水泵机组（含电机），isStandby 属性决定是否计入台数
- `Motor`：ratedPower 决定间距选取标准
- `HoistingEquipment`：trackSpan 约束净宽
- `ControlCabinet`：panelCenterHeight 需在 1.4～1.5 m

**规则（Rule）：**
- **R-LA-01** 水泵间距规则：相邻水泵净距 ≥ 1.0 m（大型 ≥ 1.2 m）
- **R-LA-02** 泵与墙壁间距规则：距墙 ≥ 0.8 m（大型 ≥ 1.0 m）
- **R-LA-03** 大型泵加大间距规则：电机功率 > 55 kW 时适用
- **R-LA-05** 进水流速规则（兼用于通道宽度参考）
- **R-OP-01** 同型号水泵规则（建议）
- **R-OP-02** 并联泵台数限制规则：N ≤ 6
- **R-OP-05** 布置对称性规则（建议）
