# AG2-1 泵房维护间尺寸计算

> 功能分类：**计算**
> 代码文件：`src/agents/ag21-maintenance-room.js`

---

## 一、职责概述

AG2-1 根据工作泵台数、备用泵台数和电机功率，计算泵房维护间的最小净尺寸（长度 L 和宽度 W），供 AG3-1 绘制二维平面图使用。

---

## 二、在系统中的位置

```
AG0-0（工作泵台数 N、备用泵台数 N_spare）→ AG2-1
AG1-2（电机功率 P_motor）                → AG2-1（接收）
                                          ▼
                              AG2-1（维护间尺寸 L、W）
                                          │
                                          ▼
                              AG3-1（设备流线二维示意）
```

---

## 三、输入参数

| 参数名 | 来源 | 单位 | 说明 |
|--------|------|------|------|
| N | AG0-0 | 台 | 工作泵台数 |
| N_spare | AG0-0 | 台 | 备用泵台数，默认 0 |
| P_motor | AG1-2 | kW | 单泵电机功率，用于判断间距规则 |

---

## 四、计算逻辑

### 4.1 设备尺寸默认值

| 参数 | 默认值 | 说明 |
|------|--------|------|
| w_pump（单泵外形宽度） | 0.6 m | 沿排列方向 |
| d_pump（单泵外形深度） | 0.8 m | 含电机联轴器，沿通道方向 |

### 4.2 间距规则（依据 P_motor）

| 条件 | 泵间净距 d_spacing | 端部距墙净距 e_wall | 依据 |
|------|--------------------|---------------------|------|
| P_motor ≤ 55 kW | 1.0 m | 0.8 m | R-LA-01、R-LA-02 |
| P_motor > 55 kW | 1.2 m | 1.0 m | R-LA-01/03（大型泵） |

### 4.3 总布置台数

```
N_total = N + N_spare
```

### 4.4 维护间净长

```
L_raw = N_total × w_pump + (N_total - 1) × d_spacing + 2 × e_wall
L = ceilTo01(L_raw)    （向上取至 0.1 m 精度）
```

**示例：** N = 2，N_spare = 0，w_pump = 0.6 m，d_spacing = 1.0 m，e_wall = 0.8 m
→ N_total = 2，L = 2×0.6 + 1×1.0 + 2×0.8 = **4.0 m**

### 4.5 维护间净宽

```
W_equip = d_pump + 0.5    （水泵深度 + 出水管附件横向空间）
W_raw   = max(1.2, W_equip) + 0.3
W       = max(2.5, ceilTo01(W_raw))
```

取 W ≥ 2.5 m（工程最小值）。

---

## 五、输出参数

| 参数名 | 单位 | 说明 |
|--------|------|------|
| L | m | 维护间净长（向上取 0.1 m） |
| W | m | 维护间净宽（≥ 2.5 m） |
| w_pump | m | 单泵外形宽度 |
| d_pump | m | 单泵外形深度 |
| d_spacing | m | 泵间净距 |
| e_wall | m | 端部距墙净距 |
| N_total | 台 | 总布置台数 |

---

## 六、代码接口

```javascript
import { runMaintenanceRoom } from '../agents/maintenance-room.js'

runMaintenanceRoom(N, motorPower, N_spare = 0)
// → { w_pump, d_pump, d_spacing, e_wall, L, W, N_total, rows }
```
