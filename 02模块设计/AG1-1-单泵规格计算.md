# AG1-1 单泵规格计算 Agent

## 一、职责概述

AG1-1 负责根据泵站总设计流量 Q、工作水泵台数 N、系统扬程参数，计算**单台水泵的设计流量、扬程、功率**，并确定**进水管和出水管的公称内径**，为水泵选型和管路设计提供完整基础参数。

---

## 二、在系统中的位置

```
AG0-0（用户对话）
  │  输入：Q（总流量）, N（台数）, 管路标高与管长参数
  ▼
AG1-1（单泵规格计算）
  │  输出：Q_single, H, P_motor, DN_inlet, DN_outlet
  ▼
后续水泵选型 / 管路设计
```

---

## 三、输入参数

| 参数名 | 符号 | 单位 | 来源 | 说明 |
|--------|------|------|------|------|
| 总设计流量 | Q | m³/h | AG0-0 | 泵站总设计流量 |
| 工作泵台数 | N | 台 | AG0-0 | 不含备用泵 |
| 停泵水位 | Stop level | m | AG2-1 | 绝对标高，由水池计算得出 |
| 出水口最高标高 | Discharge Level | m | AG0-0 | 绝对标高 |
| 出水管总长度 | L | m | AG0-0 | 用于管网阻力计算 |
| 水泵效率（预估） | η | 无量纲 | 默认 0.70 | 选型前预估值 |

---

## 四、输出参数

| 参数名 | 符号 | 单位 | 说明 |
|--------|------|------|------|
| 单泵设计流量 | Q_single | m³/h | 对应 `Pump.flowRate` |
| 系统设计扬程 | H | m | 对应 `Pump.head` |
| 轴功率 | Pz | kW | 中间计算量 |
| 电机安全系数 | K | 无量纲 | 由轴功率范围确定 |
| 电机功率 | P_motor | kW | 对应 `Pump.motorPower` |
| 进水管设计流速 | v_in | m/s | 默认 1.0 m/s |
| 进水管计算内径 | D_inner_in | mm | 连续性方程计算 |
| 进水管公称内径 | DN_inlet | mm | 向上取标准管径 |
| 出水管设计流速 | v_out | m/s | 默认 1.5 m/s |
| 出水管计算内径 | D_inner_out | mm | 连续性方程计算 |
| 出水管公称内径 | DN_outlet | mm | 向上取标准管径 |

## 五、计算逻辑

### 步骤 1：计算单泵设计流量

```
Q_single = Q / N    （m³/h）
```

- 各台水泵取相同流量（依据规则 **R-OP-01** 推荐同型号水泵）

**示例：** Q = 360 m³/h，N = 3，则 Q_single = 120 m³/h

---

### 步骤 2：计算管网阻力 R

依据设计规范 §2.2 预估管网阻力：

```
q_single = Q_single / 3600    （m³/s，用于管径公式）
D_out_approx = sqrt(4 × q_single / (π × 1.5)) × 1000    （mm，初估出水管径）
```

**Hazen-Williams 公式（每百米摩阻）：**

```
R_m = 10.67 × L_100 × q^1.852 / (C_hw^1.852 × D^4.87)    （m/100m）
```

其中：
- C_hw：Hazen-Williams 系数，钢管取 120，混凝土管取 110
- D：管径（m）
- L_100 = 100（每百米）

**各项阻力：**

| 阻力类型 | 符号 | 计算公式 |
|----------|------|----------|
| 沿程阻力 | R_L | R_m × (L / 100) |
| 局部阻力 | R_j | 0.15 × R_L |
| 设备阻力 | R_room | 2.0 m（默认） |
| **总阻力** | **R** | **R_L + R_j + R_room** |

---

### 步骤 3：计算系统扬程

```
H = (Discharge Level - Stop level) + R    （m）
```

- Discharge Level：出水口最高标高（m，绝对标高）
- Stop level：停泵水位（m，绝对标高，由 AG2-1 提供）
- R：总管网阻力（m）

**示例：** Discharge Level = 5.0 m，Stop level = -2.0 m，R = 3.5 m，则 H = 7.0 + 3.5 = **10.5 m**

---

### 步骤 4：计算水泵轴功率与电机功率

**轴功率：**

```
Pz = (Q_single × H) / (367 × η)    （kW）
```

其中 367 = 102 × 3.6（换算系数：102 将 kg·m/s 转为 kW，3.6 将 m³/hr 转为 m³/s × 3600）

**电机安全系数匹配：**

| 轴功率范围 | 安全系数 K |
|------------|------------|
| Pz < 15 kW | 1.25 |
| 15 ≤ Pz < 55 kW | 1.15 |
| Pz ≥ 55 kW | 1.10 |

**电机功率：**

```
P_motor = Pz × K    （kW）
```

**示例：** Q_single = 120 m³/h，H = 10.5 m，η = 0.70
- Pz = (120 × 10.5) / (367 × 0.70) = 1260 / 256.9 ≈ **4.90 kW**
- Pz < 15 kW → K = 1.25
- P_motor = 4.90 × 1.25 ≈ **6.13 kW**

---

### 步骤 5：计算进水管管径

流速取 v_in = **1.0 m/s**（进水管推荐流速，参考设计规范 §4.2）：

```
q_single = Q_single / 3600              （m³/s）
D_inner_in = sqrt(4 × q_single / (π × v_in)) × 1000    （mm）
```

向上取标准公称内径（标准管径系列见步骤 6），得 **DN_inlet**。

**示例：** Q_single = 120 m³/h，v_in = 1.0 m/s
- q_single = 120 / 3600 = 0.0333 m³/s
- D_inner_in = sqrt(4 × 0.0333 / (π × 1.0)) × 1000 = 206 mm
- DN_inlet = **DN250**

---

### 步骤 6：计算出水管管径

流速取 v_out = **1.5 m/s**（出水管推荐流速，参考设计规范 §4.2）：

```
D_inner_out = sqrt(4 × q_single / (π × v_out)) × 1000    （mm）
```

向上取标准公称内径，得 **DN_outlet**。

**标准公称内径系列 DN（mm）：**

```
100, 125, 150, 200, 250, 300, 350, 400, 450, 500, 600, 700, 800 , 900, 1000, 1200, 1250, 1500, 等
```

选取原则：取系列中**不小于计算内径的最小值**。

**示例：** Q_single = 120 m³/h，v_out = 1.5 m/s
- D_inner_out = sqrt(4 × 0.0333 / (π × 1.5)) × 1000 = 168 mm
- DN_outlet = **DN200**

---

### 步骤 7：验证工作点效率

确认所选水泵在计算工作点下，运行效率满足规则 **R-HY-03（水泵工作点效率规则）**：

```
η_operating ≥ 0.85 × η_BEP
```

- η_BEP：水泵最高效率点（BEP）效率，来自 `PumpCharacteristicCurve.bestEfficiencyPoint`
- 若实际工作点偏离高效区，应调整 N 或更换水泵型号

## 八、引用的本体对象与规则

**对象（Object）：**
- `Pump`：flowRate（单泵流量）、head（扬程）、motorPower（电机功率）、efficiency（效率）
- `InletPipe`：diameter（进水管管径）
- `OutletPipe`：diameter（出水管管径）
- `PumpCharacteristicCurve`：bestEfficiencyPoint（最高效率工作点）

**规则（Rule）：**
- **R-HY-02** 系统总扬程计算规则：管路流速范围 0.8～1.5 m/s
- **R-HY-03** 水泵工作点效率规则：η_operating ≥ 0.85 × η_BEP
- **R-OP-01** 同型号水泵规则：建议各台水泵采用相同型号（Q_single 相同）
- **R-OP-02** 并联泵台数限制规则：工作泵台数 N ≤ 6
