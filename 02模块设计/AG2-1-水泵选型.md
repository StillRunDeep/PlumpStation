# AG2-1 水泵选型

> 功能分类：**数值计算**
> 本体对象与共享常量定义见 [00-计算定义总纲.md](./00-计算定义总纲.md)

---

## 一、职责概述

AG2-1 根据单泵流量、管路参数及污水池停泵水位，计算**系统扬程和电机功率**，为水泵选型提供完整的技术参数。

---

## 二、在系统中的位置

```
AG0-0（用户对话）
  │  Q_single（= Q/N，AG0-0 直接计算）
  │  DischargeLevel, L
AG1-1（污水池计算）
  │  stopLevel
  ▼
AG2-1（水泵选型）
  │  输出：H, P_motor
  ▼
AG2-2（管道尺寸计算）
AG0-1（连接关系配置器）
AG3-1（设备流线二维示意）
```

---

## 三、输入参数

| 参数名 | 符号 | 单位（SI） | 来源 | 说明 |
|--------|------|-----------|------|------|
| 单泵设计流量 | Q_single | m³/h | AG0-0 | = Q / N |
| 出水口最高标高 | DischargeLevel | m | AG0-0 | 绝对标高，最终排放点 |
| 出水管总长度 | L | m | AG0-0 | 用于管网阻力计算 |
| 停泵水位 | stopLevel | m | AG1-1 | 绝对标高，污水池停泵水位 |
| 水泵效率（预估）| η | 无量纲 | 默认 0.70 | 选型前预估值 |

---

## 四、计算逻辑

### 步骤 1：计算出水管初估管径及管网阻力

```
q_single = Q_single / 3600    （m³/s）

D_out_approx = sqrt(4 × q_single / (π × 1.5)) × 1000    （mm，初估出水管径）
```

**Hazen-Williams 每百米摩阻：**

```
R_m = 10.67 × 100 × q^1.852 / (C_hw^1.852 × D^4.87)    （m/100m）
```

- C_hw：管材系数，见总纲 §4.4
- D：管径（m）

**各项阻力：**

| 阻力类型 | 符号 | 计算公式 |
|----------|------|----------|
| 沿程阻力 | R_L | R_m × (L / 100) |
| 局部阻力 | R_j | 0.15 × R_L |
| 设备阻力 | R_room | 2.0 m（默认） |
| **总阻力** | **R** | **R_L + R_j + R_room** |

---

### 步骤 2：计算系统扬程

```
H = (DischargeLevel - stopLevel) + R    （m）
```

- DischargeLevel：出水口最高标高（m，绝对标高）
- stopLevel：停泵水位（m，绝对标高，由 AG1-1 提供）
- R：总管网阻力（m）

**示例：** DischargeLevel = 5.0 m，stopLevel = −2.0 m，R = 3.5 m
→ H = (5.0 − (−2.0)) + 3.5 = **10.5 m**

---

### 步骤 3：计算轴功率与电机功率

**轴功率：**

```
Pz = (Q_single × H) / (367 × η)    （kW）
```

换算说明：367 = 102 × 3.6（102 将 kg·m/s 转为 kW；3.6 将 m³/h 转为 m³/s × 3600）

**电机安全系数：** 见总纲 §4.2

**电机功率：**

```
P_motor = Pz × K    （kW）
```

**示例：** Q_single = 120 m³/h，H = 10.5 m，η = 0.70
- Pz = (120 × 10.5) / (367 × 0.70) ≈ **4.90 kW**
- Pz < 15 kW → K = 1.25
- P_motor = 4.90 × 1.25 ≈ **6.13 kW**

---

### 步骤 4：效率验证（占位）

确认所选水泵工作点效率满足：

```
η_operating ≥ 0.85 × η_BEP
```

> **⚠ 当前为占位实现**：尚无真实泵特性曲线数据。η_operating = 0.70、η_BEP = 0.80 均为常数。
> 待 PDF 选型资料（Pump catalogue）整合后，替换为实际 Q-H-η 曲线查取值。

## 五、输出参数

| 参数名 | 符号 | 单位（SI） | 说明 |
|--------|------|-----------|------|
| 系统设计扬程 | H | m | 水泵选型依据 |
| 轴功率 | Pz | kW | 中间量 |
| 电机安全系数 | K | 无量纲 | 由 Pz 范围确定 |
| 电机功率 | P_motor | kW | 传给 AG2-1、AG2-2 |

---

## 六、引用规则

| 规则 ID | 简述 |
|---------|------|
| R-HY-02 | 系统总扬程 = 静扬程 + 管网阻力；管路流速 0.8～1.5 m/s |
| R-HY-03 | 水泵工作点效率 ≥ 0.85 × η_BEP |
| R-OP-01 | 建议各台水泵采用相同型号 |
| R-OP-02 | 工作泵台数 N ≤ 6 |

---

## 七、传递给下游的参数

```json
{
  "H": "<系统设计扬程，m>",
  "P_motor": "<电机功率，kW>"
}
```

下游接收方：AG2-2（Q_single 来自 AG0-0，不经本模块传递）、AG0-1（P_motor）、AG3-1（P_motor）
