# AG2-3 小型排水泵（SUP）

> 功能分类：**数值计算（独立分支）**
> 合同依据：Contract DC/2024/04 — Clause 41.57
> 本体对象与共享常量定义见 [00-计算定义总纲.md](./00-计算定义总纲.md)

---

## 一、职责概述

AG2-3 计算集水池内小型排水泵（Sump/Utility Pump，简称 SUP）的性能参数和管道尺寸。

小型排水泵用于以下场景：
- 主泵停机时的集水池排干（de-sump）
- 低水位时的持续应急排水
- 检修期间的降水作业

**本模块独立于主流程**，不影响 AG0-0 至 AG3-1 的主泵计算链。输出结果在 AG3-1 的独立展示区单独显示。

## 四、计算逻辑

### 步骤 1：轴功率计算

```
q_sup = Q_sup / 3600    （m³/s = 54 / 3600 = 0.015 m³/s = 15 L/s）

Pz = (Q_sup × H_sup) / (367 × η_sup)    （kW）
```

η_sup 取预估值 0.65（SUP 为小型潜水泵，效率低于主泵）：

```
Pz = (54 × 27) / (367 × 0.65) ≈ 6.12 kW
```

### 步骤 2：电机功率（含安全系数）

依据总纲 §4.2 电机安全系数表：

| 轴功率范围 | K |
|----------|---|
| Pz < 15 kW | 1.25 |

```
P_motor_sup = Pz × K = 6.12 × 1.25 ≈ 7.65 kW    → 取标准规格 7.5 kW 或 11 kW
```

### 步骤 3：管道尺寸计算

合同要求：水泵进出口流体速度 ≤ 3.6 m/s（Clause 41.57 引用 41.56 通用条款）。

取设计流速 v = 1.5 m/s（压水管）和 v = 1.0 m/s（吸水管），上限不超过 3.6 m/s：

**吸水管（SUP Inlet）：**

```
D_in = sqrt(4 × q_sup / (π × 1.0)) × 1000 = sqrt(4 × 0.015 / π) × 1000 ≈ 138 mm
DN_sup_in = DN150    （取 DN 标准系列中 ≥ 138 mm 的最小值）
```

**压水管（SUP Outlet）：**

```
D_out = sqrt(4 × q_sup / (π × 1.5)) × 1000 = sqrt(4 × 0.015 / (π × 1.5)) × 1000 ≈ 113 mm
DN_sup_out = DN125    （取 DN 标准系列中 ≥ 113 mm 的最小值）
```

验证流速（以 DN125 为例）：
```
v_check = q_sup / (π × (0.125/2)²) = 0.015 / 0.01227 ≈ 1.22 m/s    ✅ ≤ 3.6 m/s
```

## 六、泵型技术特征（合同要求）

| 项目 | 要求 |
|------|------|
| 型式 | 便携式（Portable），单级直联潜水离心泵 |

| 电机 | 鼠笼转子感应型，IEC 60034 Class |
| 控制模式 | SCADA-Auto / SCADA-Manual / Manual-Local / Manual-Field |
| 材质 | 泵壳 SS316 / 叶轮双相不锈钢 / 轴 SS316（参照主泵 Clause 41.56） |
