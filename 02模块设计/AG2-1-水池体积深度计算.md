# AG2-1 水池体积深度计算 Agent

## 一、职责概述

AG2-1 负责根据单泵设计流量和集水池占地面积，计算**集水池最小有效容积**和**有效水深**，同时确定**水泵设计扬程**。计算结果确保：

1. 集水池容积足够大，避免水泵频繁启停（保护电机寿命）
2. 集水池有效水深满足水泵淹没要求（防止汽蚀）
3. 水泵扬程覆盖实际管路阻力，含安全余量

---

## 二、在系统中的位置

```
AG0-0（用户对话）
  │  输入：Q、N、S
  ▼
AG1-1 → Q_single
  │
AG2-1（水池体积深度计算）
  │  输入：Q_single（单泵流量）, N（台数）, S（占地面积）
  │  输出：V_min（最小有效容积）, h_pool（集水池深度）, H（水泵扬程）
  ▼
泵站结构设计 / 水泵选型
```

---

## 三、输入参数

| 参数名 | 类型 | 单位 | 来源 | 说明 |
|--------|------|------|------|------|
| Q_single | Float | m³/h | AG1-1 | 单台工作水泵的设计流量 |
| N | Int | 台 | AG0-0 | 工作水泵台数 |
| S | Float | m² | AG0-0 | 集水池占地平面面积 |
| Z | Int | 次/h | 默认值/用户指定 | 水泵每小时最大允许启动次数，默认取 6（电机保护约束，范围 4～6） |
| h_static | Float | m | 用户提供/估算 | 静扬程（集水池最低工作水位至出水管排放点的高差） |

---

## 四、输出参数

| 参数名 | 类型 | 单位 | 说明 |
|--------|------|------|------|
| V_min | Float | m³ | 集水池最小有效调节容积，对应 `CollectionPool.effectiveVolume` |
| h_active | Float | m | 集水池有效调节水深（对应启泵水位与停泵水位之差） |
| h_pool | Float | m | 集水池总有效深度（含保护水深和超高） |
| H | Float | m | 水泵设计扬程，对应 `Pump.head` |

## 五、计算逻辑

### 步骤 1：计算集水池最小有效容积

依据规则 **R-CM-01（集水池有效容积规则）**，当来水量等于单泵流量一半时，水泵启停最频繁，此时所需调节容积最大：

```
V_min = Q_single / (4 × Z)
```

- Q_single：单泵流量（m³/h）
- Z：每小时最大允许启动次数（次/h），由 `Motor.maxStartsPerHour` 确定，取 4～6
- 计算结果单位为 m³

**推导说明：**

在最不利工况（来水量 = Q_single / 2）下，设调节容积为 V，则：
- 充水时间 = 2V / Q_single（h）
- 排水时间 = 2V / Q_single（h）
- 每次启停循环 = 4V / Q_single（h）
- 每小时启动次数 = Q_single / (4V) ≤ Z

因此：V ≥ Q_single / (4Z)

**示例：** Q_single = 120 m³/h，Z = 6
```
V_min = 120 / (4 × 6) = 5.0 m³
```

---

### 步骤 2：计算集水池有效调节水深

```
h_active = V_min / S
```

h_active 即为停泵水位（`CollectionPool.stopLevel`）到启泵水位（`CollectionPool.startLevel`）的高差。

**示例：** V_min = 5.0 m³，S = 4.0 m²
```
h_active = 5.0 / 4.0 = 1.25 m
```

---

### 步骤 3：计算集水池总有效深度

总深度由以下各部分叠加：

```
h_pool = h_submersion + h_active + h_reserve + h_freeboard
```

| 分项 | 符号 | 取值说明 |
|------|------|----------|
| 水泵进口淹没深度 | h_submersion | ≥ 0.5 m（依据 R-CM-03，防止吸入空气） |
| 有效调节水深 | h_active | 由步骤 2 计算 |
| 底部沉淀区深度 | h_reserve | ≥ 0.3 m（集水池底坡 ≥ 1:10，依据 R-CM-05） |
| 超高（报警水位以上余量） | h_freeboard | ≥ 0.3 m（超高保护） |

```
h_pool = 0.5 + h_active + 0.3 + 0.3
       = h_active + 1.1 m（最小值）
```

**示例：** h_active = 1.25 m
```
h_pool = 1.25 + 1.1 = 2.35 m → 向上取至 2.4 m
```

---

### 步骤 4：设定集水池三水位

依据规则 **R-CM-04（水位逻辑顺序规则）**：

```
stopLevel < startLevel < alarmLevel
```

以集水池底部为基准（= 0 m）：

```
stopLevel  = h_submersion = 0.5 m
startLevel = stopLevel + h_active
alarmLevel = startLevel + h_freeboard
```

**示例：**
```
stopLevel  = 0.5 m
startLevel = 0.5 + 1.25 = 1.75 m
alarmLevel = 1.75 + 0.3  = 2.05 m
```

---

### 步骤 5：计算水泵设计扬程

依据规则 **R-HY-02（系统总扬程计算规则）**：

```
H = (H_static + H_friction + H_local) × K_safety
```

| 分项 | 符号 | 说明 |
|------|------|------|
| 静扬程 | H_static | 集水池最低工作水位（stopLevel）至出水排放点高差；静扬程 ≈ h_pool + 出水口超高 |
| 沿程摩擦损失 | H_friction | 达西-韦斯巴赫公式计算，或按管长的 2%～5% 估算 |
| 局部阻力损失 | H_local | 弯头、止回阀（CheckValve）、闸阀（GateValve）等之和，通常取 H_friction 的 30%～50% |
| 安全系数 | K_safety | 1.05 ～ 1.10（即留 5%～10% 余量） |

**估算方法（当管路损失未知时）：**

```
H_friction_est = 管路总长 × 单位长度损失（约 0.002 ～ 0.005 m/m）
H_local_est    = H_friction_est × 0.4
H = (H_static + H_friction_est + H_local_est) × 1.08
```

**示例：** h_pool = 2.4 m，出水口超高 1.0 m，管路总长 50 m，单位损失 0.003 m/m
```
H_static    = 2.4 + 1.0 = 3.4 m
H_friction  = 50 × 0.003 = 0.15 m
H_local     = 0.15 × 0.4 = 0.06 m
H           = (3.4 + 0.15 + 0.06) × 1.08 = 3.61 × 1.08 ≈ 3.9 m
```

---

### 步骤 6：验证汽蚀安全性

依据规则 **R-HY-01（水泵安装高程汽蚀规则）** 和 **R-CM-03（停泵水位淹没深度规则）**：

```
(CollectionPool.stopLevel - Pump.inletElevation) ≥ 0.5 m
```

当停泵水位 = 0.5 m（基准），水泵进口应安装在 stopLevel 以下，即水泵进口高程 ≤ 0 m（池底以下或与池底持平）。

---

## 六、梯级启泵扩展（多泵配置）

当 N > 1 时，依据规则 **R-CL-01（梯级启泵规则）**，各台水泵启动水位应依次递增：

```
Pump_1.startLevel = stopLevel + h_active
Pump_2.startLevel = Pump_1.startLevel + Δh    （Δh ≥ 0.1 m，建议 0.2 m）
Pump_3.startLevel = Pump_2.startLevel + Δh
...
ALL.stopLevel = stopLevel（统一停泵水位）
```

## 八、引用的本体对象与规则

**对象（Object）：**
- `CollectionPool`：effectiveVolume（有效容积）、startLevel（启泵水位）、stopLevel（停泵水位）、alarmLevel（报警水位）、bottomSlope（底坡）
- `Pump`：head（扬程）、startupCount（启动次数）、installElevation（安装高程）
- `Motor`：maxStartsPerHour（最大启动次数/h）
- `CheckValve`：出水管止回阀（计入局部损失）
- `GateValve`：出水管闸阀（计入局部损失）
- `SystemResistanceCurve`：staticHead（静扬程）、resistanceCoefficient（阻力系数）

**规则（Rule）：**
- **R-CM-01** 集水池有效容积规则：V ≥ Q_single / (4Z)
- **R-CM-02** 水泵启动频率规则：启动次数 ≤ Motor.maxStartsPerHour
- **R-CM-03** 停泵水位淹没深度规则：停泵水位高于水泵进口 ≥ 0.5 m
- **R-CM-04** 水位逻辑顺序规则：stopLevel < startLevel < alarmLevel
- **R-CM-05** 集水池底部坡度规则：底坡 ≥ 1:10
- **R-HY-01** 水泵安装高程（汽蚀）规则
- **R-HY-02** 系统总扬程计算规则：H = (H_static + H_friction + H_local) × K_safety
- **R-CL-01** 梯级启泵规则：多泵按水位递增依次启动
