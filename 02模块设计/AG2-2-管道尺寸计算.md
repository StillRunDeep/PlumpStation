# AG2-2 管道尺寸计算

> 功能分类：**数值计算**
> 管道命名约定与 DN 标准系列见 [00-计算定义总纲.md](./00-计算定义总纲.md)

---

## 一、职责概述

AG2-2 根据单泵流量和总流量，按设计流速计算系统内三类管道的公称内径（DN），分别是水泵吸水管（PumpInlet）、水泵压水管（PumpOutlet）和总出水管（MainOutlet）。

---

## 二、在系统中的位置

```
AG2-1（水泵选型）
  │  Q_single
AG0-0（用户对话）
  │  Q（总流量）, N（泵数）
  ▼
AG2-2（管道尺寸计算）
  │  输出：DN_pumpIn, DN_pumpOut, DN_mainOutlet
  ▼
AG0-1（连接关系配置器）
AG3-1（设备流线二维示意）
```

---

## 三、输入参数

| 参数名 | 符号 | 单位（SI） | 来源 | 说明 |
|--------|------|-----------|------|------|
| 单泵设计流量 | Q_single | m³/h | AG2-1 | 每台工作泵的流量 |
| 泵站总流量 | Q | m³/h | AG0-0 | 用于计算总出水管 |
| 工作泵台数 | N | 台 | AG0-0 | 校验 Q_single × N = Q |

---

## 四、计算逻辑

### 通用公式

将流量由 m³/h 转换为 m³/s，由连续性方程计算管径：

```
q = Q_vol / 3600    （m³/s）
D_calc = sqrt(4 × q / (π × v)) × 1000    （mm）
```

再从 DN 标准系列（见总纲 §4.1）中取**不小于 D_calc 的最小值**，得公称内径 DN。

---

### 步骤 1：计算 PumpInlet 管径（吸水管，每台泵独立）

设计流速 v_in = **1.0 m/s**（防止流速过高引起汽蚀）：

```
q_single = Q_single / 3600    （m³/s）
D_pumpIn = sqrt(4 × q_single / (π × 1.0)) × 1000    （mm）
DN_pumpIn = ceil_to_DN_series(D_pumpIn)
```

**示例：** Q_single = 120 m³/h
- q_single = 0.0333 m³/s
- D_pumpIn = sqrt(4 × 0.0333 / π) × 1000 = 206 mm
- DN_pumpIn = **DN250**

---

### 步骤 2：计算 PumpOutlet 管径（压水管，每台泵独立）

设计流速 v_out = **1.5 m/s**：

```
D_pumpOut = sqrt(4 × q_single / (π × 1.5)) × 1000    （mm）
DN_pumpOut = ceil_to_DN_series(D_pumpOut)
```

**示例：** Q_single = 120 m³/h
- D_pumpOut = sqrt(4 × 0.0333 / (π × 1.5)) × 1000 = 168 mm
- DN_pumpOut = **DN200**

---

### 步骤 3：计算 MainOutlet 管径（总出水干管）

总出水管承载所有工作泵合流后的总流量 Q：

```
q_total = Q / 3600    （m³/s）
D_mainOutlet = sqrt(4 × q_total / (π × 1.5)) × 1000    （mm）
DN_mainOutlet = ceil_to_DN_series(D_mainOutlet)
```

**示例：** Q = 360 m³/h（N = 3 台，Q_single = 120 m³/h）
- q_total = 360 / 3600 = 0.1 m³/s
- D_mainOutlet = sqrt(4 × 0.1 / (π × 1.5)) × 1000 = 291 mm
- DN_mainOutlet = **DN300**

## 五、输出参数

| 参数名 | 符号 | 单位（SI） | 说明 |
|--------|------|-----------|------|
| 吸水管公称内径 | DN_pumpIn | mm | 每台泵独立吸水管 |
| 压水管公称内径 | DN_pumpOut | mm | 每台泵独立压水管 |
| 总出水管公称内径 | DN_mainOutlet | mm | 汇总干管 |

## 六、引用规则

| 规则 ID | 简述 |
|---------|------|
| R-HY-02 | 管路流速范围：进水管 0.8～1.2 m/s，出水管 1.0～1.8 m/s |
| R-LA-04 | 进水管独立取水规则：每台泵独立吸水管 |

## 七、传递给下游的参数

```json
{
  "DN_pumpIn": "<吸水管公称内径，mm>",

  "DN_mainOutlet": "<总出水管公称内径，mm>"
}
```

下游接收方：AG0-1（节点规格配置）、AG3-1（管道绘图宽度）
