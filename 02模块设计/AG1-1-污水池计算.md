# AG1-1 调蓄池计算

> 功能分类：**数值计算**
> 依据：香港渠务署《雨水排水手册（第五版）》第14章
> 英文对象名：`WetWell`
> 代码文件：`src/agents/ag21-pool-depth.js`

---

## 一、职责概述

AG1-1 根据设计流量和集水池几何参数，计算：
1. **调蓄池容积**（步骤3）
2. **控制水位**（停泵水位、启泵水位、最高水位、报警水位）（步骤5）

---

## 二、在系统中的位置

```
AG0-0（用户对话）
  │  Q, Q_single, Q_p, N, S, Z_bottom, Z_top
  ▼
AG1-1（调蓄池计算）
  │  输出：V_required, Z_stop, Z_start1, Z_start2, Z_max, Z_alarm
  ▼
AG2-1（水泵选型，使用 Z_stop 计算扬程）
AG3-1（设备流线二维示意，使用三水位绘制剖面图）
```

### 水位关系校验规则

| 校验条件 | 依据 |
|---------|------|
| Z_stop ≥ Z_bottom | 停泵水位不得低于池底 |
| Z_start1 > Z_stop | 1#泵启动水位应高于停泵水位 |
| Z_start2 > Z_start1 | 2#泵启动水位应高于1#泵启动水位 |
| Z_max < Z_top - F_b | 最高水位应低于池顶减去超高 |
| Z_alarm_low ≥ Z_bottom | 低水位报警不得低于池底 |

## 五、计算逻辑

### 步骤 3：调蓄池容积计算

**依据**：手册第14.5.2节

#### 3.1 单泵流量

```
Q_pump = Q_single / 3600    （m³/s）
```

#### 3.2 削峰容积需求

```
V_required = (Q_p - Q_pump) × t_peak × 60 × k_shape    （m³）
```

其中：
- Q_p = 峰值流量（m³/s）
- Q_pump = 单泵流量（m³/s）
- t_peak = 暴雨峰现时间（min），通常取 60-120 min
- k_shape = 暴雨剖面形状系数，通常 0.5-0.75

#### 3.3 有效调蓄深度

```
h_active = V_required / S    （m）
```

#### 3.4 总池深

```
D = Z_top - Z_bottom    （m）
```

---

### 步骤 5：控制水位设计

**依据**：手册第6.5节、第14.6.1节

#### 5.1 集水坑底标高

```
Z_sump = Z_bottom - Z_sump_offset    （mPD）
```

#### 5.2 停泵水位（Pump Stop Level）

```
Z_stop = Z_sump + h_sub    （mPD）
```

其中 h_sub = 停泵淹没深度（默认0.5m）

**限制条件**：
```
Z_stop ≥ Z_bottom
```

#### 5.3 1#泵启动水位（First Pump Start Level）

```
Z_start1 = Z_stop + k1 × D    （mPD）
```

其中 k1 = 0.40-0.50（40-50%池深，对应约40-50%容积）

#### 5.4 2#泵启动水位（Second Pump Start Level）

```
Z_start2 = Z_stop + k2 × D    （mPD）
```

其中 k2 = 0.75-0.85（75-85%池深）

#### 5.5 最高水位（Maximum Water Level）

```
Z_max = Z_top - F_b - F_s    （mPD）
```

其中：
- F_b = 超高（默认0.8m，手册第6.5节）
- F_s = 安全余量（默认0.3m）

**限制条件**：
```
Z_max ≤ Z_top - 0.8 m    （超高验证）
```

#### 5.6 报警水位（Alarm Levels）

```
Z_alarm_low = Z_stop - 0.3    （mPD，低水位报警）
Z_alarm_high = Z_max - 0.3    （mPD，高水位报警）
```

---

### 多泵梯级启动水位（N > 2 时）

依据规则 R-CL-01（多泵梯级启动规则）：

```
Pump_1.startLevel = Z_start1
Pump_2.startLevel = Z_start2
Pump_3.startLevel = Z_start2 + Δh    （Δh ≥ 0.2 m）
Pump_n.startLevel = Z_start(n-1) + Δh
所有泵统一停泵水位 = Z_stop
```

## 八、示例计算

**已知**：Q=1872 m³/h, Q_single=936 m³/h, Q_p=0.52 m³/s, N=2, S=500 m², Z_bottom=-10.0 mPD, Z_top=5.0 mPD

**设计参数**：
- t_peak = 90 min（手册第14.5.2节）
- k_shape = 0.6（手册第14.5.2节）
- h_sub = 0.5 m（工程惯例）
- k1 = 0.45（手册第14.6.1节）
- k2 = 0.80（手册第14.6.1节）
- F_b = 0.8 m（手册第6.5节）
- F_s = 0.3 m（工程惯例）
- Z_sump_offset = 2.5 m（工程惯例）

**步骤 3**：
```
Q_pump = 936/3600 = 0.26 m³/s
V_required = (0.52 - 0.26) × 90 × 60 × 0.6 = 842 m³
h_active = 842/500 = 1.68 m
D = 5.0 - (-10.0) = 15.0 m
```

**步骤 5**：
```
Z_sump = -10.0 - 2.5 = -12.5 mPD
Z_stop = -12.5 + 0.5 = -12.0 mPD
Z_start1 = -12.0 + 0.45×15.0 = -5.25 mPD
Z_start2 = -12.0 + 0.80×15.0 = 0.0 mPD
Z_max = 5.0 - 0.8 - 0.3 = 3.9 mPD
```
