# AG1-1 污水池计算

> 功能分类：**数值计算**
> 英文对象名：`WetWell`
> 本体对象与共享常量定义见 [00-计算定义总纲.md](./00-计算定义总纲.md)

---

## 一、职责概述

AG1-1 根据单泵流量和污水池几何约束，计算**最小有效调节容积、有效水深、三水位控制逻辑**，确保水泵不频繁启停且满足淹没保护要求。

支持**双模式输入**：用户可按已知面积或已知深度任选其一。

---

## 二、在系统中的位置

```
AG0-0（用户对话）
  │  Q_single（= Q/N，AG0-0 直接计算输出）
  │  N, Z, S 或 h_wetWell（用户选择模式）
  ▼
AG1-1（污水池计算）
  │  输出：V_min, h_wetWell, S_min, stopLevel, startLevel, alarmLevel
  ▼
AG2-1（水泵选型，使用 stopLevel 计算扬程）
AG3-1（设备流线二维示意，使用三水位绘制剖面图）
```

| 参数名 | 符号 | 单位（SI） | 来源 | 说明 |
|--------|------|-----------|------|------|

### 模式 B：已知深度

| 参数名 | 符号 | 单位（SI） | 说明 |
|--------|------|-----------|------|
| 污水池总深度 | h_wetWell | m | 用户输入，由场地/结构约束确定 |

**计算目标**：所需最小平面面积 S_min

## 四、计算逻辑

### 步骤 1：计算最小有效调节容积（两种模式通用）

依据最不利工况（来水量 = Q_single / 2 时启停最频繁）：

```
V_min = Q_single / (4 × Z)    （m³）
```

**推导：** 在来水量 = Q_single/2 时，设调节容积为 V：
- 充水时间 = 2V / Q_single，排水时间 = 2V / Q_single
- 循环周期 = 4V / Q_single，启动次数 = Q_single / (4V) ≤ Z
- → V ≥ Q_single / (4Z)

**示例：** Q_single = 120 m³/h，Z = 6 → V_min = 120 / 24 = **5.0 m³**

---

### 步骤 2a（模式 A）：由面积 S 推算深度

```
h_active = V_min / S    （m）

h_wetWell = h_active + 1.1    （m，固定分项合计见总纲 §4.5）
```

**示例：** V_min = 5.0 m³，S = 4.0 m²
- h_active = 5.0 / 4.0 = 1.25 m
- h_wetWell = 1.25 + 1.1 = **2.35 m → 向上取至 2.4 m**

---

### 步骤 2b（模式 B）：由深度 h_wetWell 推算最小面积

```
h_active = h_wetWell - 1.1    （m）

S_min = V_min / h_active    （m²）
```

若 h_active ≤ 0，说明输入深度不足，系统报错提示最小所需深度。

**示例：** V_min = 5.0 m³，h_wetWell = 2.4 m
- h_active = 2.4 − 1.1 = 1.3 m
- S_min = 5.0 / 1.3 ≈ **3.85 m²**

---

### 步骤 3：设定三水位（两种模式通用）

以池底为基准（= 0 m），依据水位逻辑顺序规则：

```
stopLevel  = h_sub = 0.5 m
startLevel = stopLevel + h_active
alarmLevel = startLevel + h_freeboard = startLevel + 0.3 m
```

水位顺序验证：stopLevel < startLevel < alarmLevel

**示例（h_active = 1.25 m）：**
```
stopLevel  = 0.5 m
startLevel = 0.5 + 1.25 = 1.75 m
alarmLevel = 1.75 + 0.3 = 2.05 m
```

---

### 步骤 4：多泵梯级启动水位（N > 1）

各台水泵启动水位依次递增（梯级启泵规则）：

```
Pump_1.startLevel = stopLevel + h_active
Pump_2.startLevel = Pump_1.startLevel + Δh    （Δh ≥ 0.1 m，建议 0.2 m）
Pump_3.startLevel = Pump_2.startLevel + Δh
...
所有泵统一停泵水位 = stopLevel
```

---

### 步骤 5：汽蚀安全验证

```
stopLevel - Pump.inletElevation ≥ 0.5 m
```

停泵水位（0.5 m）高于水泵进口高程至少 0.5 m，即水泵进口高程 ≤ 0 m（池底处或以下）。

## 五、输出参数

| 参数名 | 符号 | 单位（SI） | 说明 |
|--------|------|-----------|------|
| 最小有效容积 | V_min | m³ | — |
| 有效调节水深 | h_active | m | stopLevel 到 startLevel 高差 |
| 污水池总深度 | h_wetWell | m | 模式 A 计算结果；模式 B 为用户输入 |
| 所需最小面积 | S_min | m² | 模式 B 计算结果；模式 A 为用户输入 |
| 停泵水位 | stopLevel | m | 距池底，传给 AG1-2 用于扬程计算 |
| 首台启泵水位 | startLevel | m | 距池底 |
| 报警水位 | alarmLevel | m | 距池底，传给 AG2-2 绘剖面图 |

---

## 六、引用规则

| 规则 ID | 简述 |
|---------|------|
| R-CM-01 | 有效容积 V ≥ Q_single / (4Z) |
| R-CM-02 | 水泵启动次数 ≤ maxStartsPerHour |
| R-CM-03 | 停泵水位高于水泵进口 ≥ 0.5 m |
| R-CM-04 | 水位顺序：stopLevel < startLevel < alarmLevel |
| R-CM-05 | 池底坡度 ≥ 1:10 |
| R-HY-01 | 水泵安装高程汽蚀规则 |
| R-CL-01 | 多泵梯级启动：各泵 startLevel 依次递增 |
