# AG0-0 用户对话 Agent

> 功能分类：**用户交互 + 数值计算**
> 依据：香港渠务署《雨水排水手册（第五版）》第4章、第7章
> 代码文件：`src/agents/ag00-validate.js`

---

## 一、职责概述

AG0-0 是雨水泵站智能设计系统的**入口 Agent**，负责：
1. 通过界面输入收集设计参数
2. **执行暴雨分析计算（步骤1）**：根据重现期和暴雨历时计算降雨强度
3. **执行径流估算计算（步骤2）**：根据汇水区特性计算设计流量 Q
4. 参数合法性校验
5. 将计算结果传递给下游 Agent

---

## 四、计算逻辑

### 步骤 1：暴雨分析（IDF 公式）

**依据**：手册第4.3节

根据用户选择的分区和重现期，从常数表（见 00-计算定义总纲 §6.1）获取 IDF 常数 a, b, c：

```
i = a / (t_d + b)^c    （mm/h）
```

**IDF常数表（手册表3a-3d）**：

| 分区 | 重现期 | a | b | c |
|------|--------|---|---|---|
| 香港天文台总部 (tungkl) | 10年 | 14.610 | 13.055 | 0.741 |
| 香港天文台总部 (tungkl) | 50年 | 17.903 | 14.896 | 0.732 |
| 香港天文台总部 (tungkl) | 200年 | 20.558 | 16.150 | 0.725 |
| 大帽山 (tai-mo-shan) | 10年 | 16.610 | 13.055 | 0.741 |
| 大帽山 (tai-mo-shan) | 50年 | 19.903 | 14.896 | 0.732 |
| 大帽山 (tai-mo-shan) | 200年 | 22.558 | 16.150 | 0.725 |

**示例**：大帽山分区，T=10年，t_d=120min
- 查表：a=16.610, b=13.055, c=0.741
- i = 16.610 / (120 + 13.055)^0.741 ≈ 2.08 mm/h

---

### 步骤 2：径流估算（推理法）

**依据**：手册第7.5.2节

```
Q_p = 0.278 × C × i × A × ARF    （m³/s）
```

**转换为 m³/h 用于后续计算**：
```
Q = Q_p × 3600    （m³/h）
```

**面积折减系数**：
```
ARF = 1.00                        （A ≤ 25 km²）
ARF = 1.547 / (A + 280.11)       （A > 25 km²）
```

**示例**：A=1.0km², C=0.90, i=2.08mm/h, ARF=1.0
- Q_p = 0.278 × 0.90 × 2.08 × 1.0 × 1.0 = 0.52 m³/s
- Q = 0.52 × 3600 = 1872 m³/h

---

### 步骤 3：单泵流量计算

```
Q_single = Q / N    （m³/h）
```

## 五、参数校验规则

### AG0-0 参数范围定义

| 参数 | 最小值 | 最大值 | 单位 | 依据 | 错误提示 |
|------|--------|--------|------|------|----------|
| zone | - | - | - | 手册第4.3.2节 | 请选择有效的暴雨分区 |
| T | - | - | - | 手册第14.6.2节 | 设计重现期 T 应为 10/50/200 年 |
| t_d | 10 | 240 | min | 手册第4.3.4节 | 暴雨历时 t_d 应在 10-240 min 范围内 |
| A | 0.001 | 100 | km² | 手册第7.5.2节 | 集水区面积 A 应在 0.001-100 km² 范围内 |
| C | 0.05 | 1.0 | - | 手册表7.5.2 | 径流系数 C 应在 0.05-1.0 范围内 |
| N | 1 | 6 | 台 | 手册第14.6.2节 | 工作泵台数 N 应为 1-6 之间的整数 |
| S | 10 | 100000 | m² | - | 集水池面积 S 必须大于 0 |
| Z_bottom | -30 | 10 | mPD | - | 池底标高 Z_bottom 应在 -30-10 mPD 范围内 |
| Z_top | -10 | 50 | mPD | - | 池顶标高 Z_top 应在 -10-50 mPD 范围内 |

### 水位关系校验

```
Z_bottom < Z_top  （池底标高应小于池顶标高）
```

### 警告条件

| 条件 | 警告信息 |
|------|----------|
| N > 6 | 工作泵台数超过 6 台，效率可能下降 |
| Q_single > 2000 m³/h | 单泵流量偏大，请确认输入参数 |

```json

  "t_d": "<暴雨历时，min>",
  "A": "<集水区面积，km²>",
  "C": "<径流系数>",
  "ARF": "<面积折减系数>",
  "i": "<设计暴雨强度，mm/h>",
  "Q_p": "<峰值流量，m³/s>",
  "Q": "<总设计流量，m³/h>",
  "Q_single": "<单泵设计流量，m³/h>",
  "N": "<工作水泵台数，台>",
  "N_spare": "<备用泵台数，台>",
  "S": "<集水池面积，m²>",
  "Z_bottom": "<池底标高，mPD>",
```

下游 Agent 接收方：
- **AG1-1**：接收 Q, Q_single, Q_p, N, S, Z_bottom, Z_top → 调蓄池计算
- **AG2-1**：接收 Q_single → 水泵选型
- **AG1-2**：接收 N → 维护间尺寸
