# AG3-1 设备流线二维示意

> 功能分类：**图形表达**
> 本体对象定义见 [00-计算定义总纲.md](./00-计算定义总纲.md)

---

## 一、职责概述

AG3-1 根据 AG0-1 输出的连接拓扑和最小空间规则，将所有设备摆入维护间，并生成两个视图：

- **平面图（俯视）**：设备水平布局、管道走向、阀件位置
- **A-A 剖面图**：竖向空间关系，包含污水池深度和三水位标注

维护间净尺寸（L_room、W_room）由本模块根据间距规则自动计算。

---

## 二、在系统中的位置

```
AG0-1（连接关系配置器）  → ConnectionTopology
AG2-1（水泵选型）        → P_motor
AG2-2（管道尺寸计算）    → DN_pumpIn, DN_pumpOut, DN_mainOutlet
AG1-1（污水池计算）      → h_wetWell, stopLevel, startLevel, alarmLevel
AG0-0（用户对话）        → N
  ▼
AG3-1（设备流线二维示意）
  │  输出：L_room, W_room + SVG 平面图 + A-A 剖面图
  ▼
网页展示区域
```

## 三、输入参数

| 参数名 | 来源 | 单位（SI） | 说明 |
|--------|------|-----------|------|
| ConnectionTopology | AG0-1 | — | 节点 + 边，含所有设备和管道连接关系 |
| P_motor | AG2-1 | kW | 用于判断间距规则 |
| DN_pumpIn | AG2-2 | mm | 吸水管管径（绘图参考宽度） |
| DN_pumpOut | AG2-2 | mm | 压水管管径 |
| DN_mainOutlet | AG2-2 | mm | 总出水管管径 |
| h_wetWell | AG1-1 | m | 污水池总深度（剖面图）|
| stopLevel | AG1-1 | m | 停泵水位（剖面图标注）|
| startLevel | AG1-1 | m | 启泵水位（剖面图标注）|
| alarmLevel | AG1-1 | m | 报警水位（剖面图标注）|
| N | AG0-0 | 台 | 工作泵台数 |

## 四、维护间尺寸计算

### 4.1 设备尺寸默认值

| 参数 | 默认值 | 说明 |
|------|--------|------|
| w_pump | 0.6 m | 单台水泵机组沿排列方向宽度 |
| d_pump | 0.8 m | 单台水泵机组沿通道方向深度（含电机）|

### 4.2 间距规则（依据 P_motor）

| 条件 | 泵间净距 d_spacing | 端部距墙净距 e_wall |
|------|--------------------|---------------------|
| P_motor ≤ 55 kW | 1.0 m | 0.8 m |
| P_motor > 55 kW | 1.2 m | 1.0 m |

### 4.3 维护间净长

布置 N+1 台（N 工作泵 + 1 备用泵，同规格），一字排开：

```
N_total = N + 1
L_room = N_total × w_pump + (N_total - 1) × d_spacing + 2 × e_wall    （m）
```

向上取至 0.1 m 精度。

**示例：** N = 3，N_total = 4，w_pump = 0.6 m，d_spacing = 1.0 m，e_wall = 0.8 m
→ L_room = 4×0.6 + 3×1.0 + 2×0.8 = **7.0 m**

### 4.4 维护间净宽

```
W_equip = d_pump + 0.5 m    （水泵深度 + 出水管附件横向空间）
W_room  = max(1.2, W_equip) + 0.3 m    （取维护通道最小 1.2 m 与设备宽中的大值，再加余量）
```

取 W_room ≥ 2.5 m（工程最小值）。

## 五、绘制内容

### 5.1 坐标系

| 视图 | X 轴 | Y 轴 | 说明 |
|------|------|------|------|
| 平面图 | 向右（泵排列方向）| 向下（通道方向）| 俯视 |
| A-A 剖面图 | 室内净宽 W | 向上（高程）| 竖向 |

两视图并列于同一 SVG 画布；剖面切线在平面图中以虚线及"A"符号标注。

### 5.2 平面图绘制对象

| 对象 | 绘制方式 | 位置依据 |
|------|----------|----------|
| 维护间外轮廓 | 矩形，L_room × W_room | 全局边界 |
| 污水池 | 矩形，面积 = S（或 S_min），以维护间为中心对称 | 维护间进水侧 |
| 水泵机组 × N_total | 矩形块，w_pump × d_pump | 一字排开，按 d_spacing 和 e_wall 定位 |
| 管道（按 ConnectionTopology）| 线段 | 按节点连接关系绘制，管线宽度参考 DN |
| 止回阀 × N | 菱形符号 | 每台泵 PumpOutlet 上 |
| 闸阀 × N | 矩形符号 | CheckValve 外侧 |
| MainOutlet 干管 | 横向粗线 | 紧靠维护间背墙 |
| 控制柜 | 矩形块 | 维护间端部角落 |
| 剖面切线 A-A | 水平虚线 + "A"标注 | 穿过任一水泵中心线 |
| 尺寸标注 | 双向箭头 + 文字 | L_room, W_room, d_spacing, e_wall |

### 5.3 A-A 剖面图绘制对象

泵房净高估算：

```
room_H = max(3.0, h_wetWell × 0.2 + 1.0)    （m）
```

| 要素 | 表达方式 | 说明 |
|------|----------|------|
| 室外地坪线（±0.00）| 粗实线 | 地上/地下分界 |
| 泵房围护结构 | 矩形，W_room × room_H | 地上部分 |
| 污水池池壁 | 矩形，W_room × h_wetWell | 地下部分 |
| 水面示意 | 蓝色填充 | 从池底到报警水位 |
| 停泵水位 stopLevel | 绿色水平虚线 + 标注 | 距池底 |
| 启泵水位 startLevel | 橙色水平虚线 + 标注 | 距池底 |
| 报警水位 alarmLevel | 红色水平虚线 + 标注 | 距池底 |
| 水泵机组（剖面）| 矩形 | 泵房内地坪处 |
| 吸水管（PumpInlet）| 竖向线段向下 | 穿底板进入污水池 |
| 压水管（PumpOutlet）| 竖向线段向上 | 水泵出口引出 |
| 竖向尺寸标注 | 双向箭头 + 文字 | room_H, h_wetWell |
| 水平尺寸标注 | 双向箭头 + 文字 | W_room |

### 5.4 缩放比例

```
scale_plan = min(
  plan_area_width  / L_pool,
  plan_area_height / (W_room + D_pool)
)    （px/m）

scale_section = min(
  section_area_width  / (W_room + 1.5),
  section_area_height / (room_H + h_wetWell + 0.5)
)    （px/m）
```

用户可通过鼠标滚轮缩放和拖拽平移，以及工具栏 ＋/⊙/－ 按钮操作。

## 六、引用规则

| 规则 ID | 简述 |
|---------|------|
| R-LA-01 | 相邻水泵净距 ≥ 1.0 m（大型泵 ≥ 1.2 m）|
| R-LA-02 | 泵距侧墙 ≥ 0.8 m（大型泵 ≥ 1.0 m）|
| R-LA-03 | 大型泵（P_motor > 55 kW）加大间距 |
| R-LA-04 | 每台泵独立吸水管（PumpInlet）|
| R-LA-08 | 就地控制柜布置在角落 |
| R-CM-03 | 停泵水位高于水泵进口 ≥ 0.5 m（剖面图可视验证）|
| R-CM-04 | 水位顺序：stopLevel < startLevel < alarmLevel |
| R-OP-01 | 同型号水泵，w_pump 相同 |
| R-OP-03 | 备用泵 1 台（N+1），同规格 |
| R-OP-05 | 水泵左右对称布置 |

## 七、输出

```json
{
  "L_room": "<维护间净长，m>",
  "W_room": "<维护间净宽，m>",
  "d_spacing": "<实际泵间净距，m>",
  "e_wall": "<端部距墙净距，m>",
  "svg": "<SVG 元素，嵌入网页展示>"
}
```

SVG 包含：
- 左侧平面图（俯视，设备 + 拓扑管道）
- 右侧 A-A 剖面图（竖向空间 + 三水位标注）
