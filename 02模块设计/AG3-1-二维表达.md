# AG3-1 二维表达 Agent

## 一、职责概述

AG3-1 负责将前序 Agent 的计算结果转化为**二维泵房示意图**，在网页 SVG 画布上精准绘制泵房内的主要设备对象。输出两个视图：

- **平面图（俯视）**：展示泵房内水平布局，设备尺寸与位置与计算结果一致
- **A-A 剖面图**：展示泵房竖向空间关系，包含集水池深度、水位标高信息

目标：让设计人员在获得数值结果的同时，直观看到泵房的平面布局与竖向空间。

---

## 二、在系统中的位置

```
AG1-1 → Q_single, D_nominal
AG1-2 → L（维护间净长）, W（维护间净宽）, d_spacing, e_wall, w_pump, d_pump
AG2-1 → h_pool, V_min, stopLevel, startLevel, alarmLevel
AG0-0 → N（水泵台数）
  │
  ▼
AG3-1（二维表达）
  │  输出：SVG 矢量图（平面图 + A-A 剖面图，支持缩放平移）
  ▼
网页展示区域
```

## 四、绘制内容与规则

### 4.1 坐标系与视图组织

本 Agent 输出**两个视图**，均使用 **SVG 矢量格式**，支持鼠标滚轮缩放和拖拽平移：

| 视图 | 坐标轴 | 核心内容 |
|------|--------|----------|
| 平面图（俯视） | X 轴向右（泵排列方向），Y 轴向下（通道方向） | 泵房轮廓、集水池平面、水泵机组、管线阀件 |
| A-A 剖面图 | 水平轴 = 室内净宽 W，竖向轴 = 高程（Z 轴向上） | 地上泵房结构、地下集水池、三水位线、进出水管 |

两视图并列展示在同一 SVG 画布中；A-A 切割位置在平面图中以虚线及"A"符号标注。

### 4.2 平面图绘制对象清单

| 对象 | 本体类 | 绘制方式 | 定位依据 |
|------|--------|----------|----------|
| 维护间外轮廓 | RainfallPumpStation | 矩形边框，L × W | 全局边界 |
| 集水池 | CollectionPool | 矩形，面积 = S，**宽度可大于维护间净长**，以维护间为中心对称布置 | 与维护间共享一侧墙（局部），超出部分为集水池自身结构 |
| 水泵机组 × N | Pump + Motor | 矩形块，w_pump × d_pump | 一字排开，间距 d_spacing，距墙 e_wall |
| 进水管 × N | InletPipe | 连接水泵与集水池的短线段 | 每台泵独立取水（R-LA-04） |
| 出水管 × N | OutletPipe | 从水泵延伸至汇总干管的线段 | 汇集到维护间对侧墙 |
| 出水总管 | OutletPipe（干管） | 横向干管，贯通全部水泵出口 | 紧靠维护间背墙 |
| 止回阀 × N | CheckValve | 出水支管上的符号（菱形） | 每台泵出水管上 |
| 闸阀 × N | GateValve | 出水支管上的符号（矩形） | 止回阀外侧 |
| 控制柜 | ControlCabinet | 矩形块 | 维护间端部角落（R-LA-08） |
| 剖面切线 A-A | — | 水平虚线 + "A"端点符号 | 穿过任一水泵中心线 |
| 尺寸标注 | — | 双向箭头 + 文字 | L、W、L_pool、D_pool、d_spacing、e_wall |

> **注意：集水池三水位（停泵水位、启泵水位、报警水位）为 Z 轴（竖向深度）数据，不在平面图中表达，仅在 A-A 剖面图中标注。**

### 4.3 集水池平面尺寸推算

集水池平面宽度（L_pool）与维护间净长（L）相互独立，按下式计算：

```
L_pool = max(L, sqrt(S × 1.5))    （m，保证集水池平面不过细长）
D_pool = S / L_pool                （m，集水池进深）
```

集水池以维护间为中心，向两侧各延伸 `(L_pool - L) / 2`，绘制在维护间进水侧。

### 4.4 A-A 剖面图绘制内容

A-A 剖面沿泵轴方向切开，展示泵房竖向空间（Z 轴）信息。

#### 泵房净高估算

```
room_H = max(3.0, h_pool × 1.2 + 1.0)    （m，泵房室内净高）
```

#### 剖面图绘制对象

| 要素 | 表达方式 | 说明 |
|------|----------|------|
| 室外地坪线（±0.00） | 粗实线 | 区分地上/地下部分 |
| 泵房围护结构 | 矩形实线框，W × room_H | 地上部分 |
| 集水池池壁 | 矩形实线框，W × h_pool | 地下部分 |
| 水面（示意） | 蓝色填充 | 从池底到报警水位 |
| 停泵水位 stopLevel | 绿色水平虚线 + 标注 | 距池底，Z 轴数据 |
| 启泵水位 startLevel | 橙色水平虚线 + 标注 | 距池底，Z 轴数据 |
| 报警水位 alarmLevel | 红色水平虚线 + 标注 | 距池底，Z 轴数据 |
| 水泵机组（剖面符号） | 矩形 | 位于泵房内地坪处 |
| 进水管（吸水管） | 竖向线段向下 | 从水泵进口穿底板进入集水池 |
| 出水管 | 竖向线段向上 | 从水泵出口引出至泵房顶 |
| 竖向尺寸标注 | 双向箭头 + 文字 | room_H（室高）、h_pool（池深） |
| 水平尺寸标注 | 双向箭头 + 文字 | W（室内净宽） |

### 4.5 SVG 缩放比例

SVG 画布分左（平面图）、右（剖面图）两区，各自独立计算比例尺：

```
scale_plan = min(
  plan_area_width  / L_pool,
  plan_area_height / (W + D_pool)
)

scale_section = min(
  section_area_width  / (W + 1.5),
  section_area_height / (room_H + h_pool + 0.5)
)    （px/m，SVG 内部单位）
```

用户可通过鼠标滚轮缩放、拖拽平移，以及工具栏"+"/"−"/"⊙"按钮操作整体视图。

## 五、图例与标注

- **平面图设备图例**：左上角彩色方块 + 名称
- **平面图尺寸标注**：蓝色细线 + 箭头，标注 L、W、d_spacing、e_wall
- **平面图比例尺**：横条，显示实际长度（如「── 1 m」）
- **剖面图水位标注**：三色水平虚线，右侧文字注明水位名称与高程值
- **剖面图高程标注**：左侧双向箭头，标注 room_H 与 h_pool
- **设备标签**：每台水泵编号（P1、P2…）

## 六、引用的本体对象与规则

**对象（Object）：**
- `RainfallPumpStation`：顶层容器，确定外轮廓
- `Pump`：机组矩形块，w_pump × d_pump
- `CollectionPool`：集水池矩形，面积 = S；stopLevel、startLevel、alarmLevel 在剖面图中标注
- `InletPipe`：独立进水管线段（R-LA-04）
- `OutletPipe`：出水支管 + 干管
- `CheckValve`：出水管止回阀符号
- `GateValve`：出水管闸阀符号
- `ControlCabinet`：就地控制柜矩形

**规则（Rule）：**
- **R-LA-01** 水泵间距规则：d_spacing ≥ 1.0/1.2 m → 绘图间距
- **R-LA-02** 泵与墙壁间距规则：e_wall ≥ 0.8/1.0 m → 绘图边距
- **R-LA-04** 进水管独立取水规则：每台泵独立进水管
- **R-LA-08** 就地控制面板高度规则：控制柜布置在角落
- **R-CM-03** 停泵水位淹没深度规则：剖面图中标注 stopLevel 与泵进口高差
- **R-CM-04** 水位逻辑顺序规则：stopLevel < startLevel < alarmLevel（剖面图中可视验证）
- **R-OP-05** 布置对称性规则：水泵左右对称排布

- 左侧**平面图（俯视）**：泵房水平布局
- 右侧 **A-A 剖面图**：泵房竖向空间与三水位

交互支持：
- 鼠标滚轮缩放（以鼠标位置为中心）
- 鼠标拖拽平移
- 工具栏 ＋ / ⊙ / － 按钮（放大 / 复位 / 缩小）
