# 泵房主要空间自动排布 — 实施计划

## Context

当前 `index.html` 中的 AG1-2 + AG3-1 仅实现了**单排（单列）**布置：
- `runAG12()` 按 `N × w_pump + (N-1)×d_spacing + 2×e_wall` 计算维护间净长 L
- `runAG31()` 把所有水泵一字排开，集水池居中置于维护间进水侧

缺失的能力：
- `layoutType` 字段在本体中已预留（单排 / 双排对置 / 地下湿式），但从未被使用
- 台数多（N > 4）时应自动切换为双排对置，缩短泵房长度
- 控制柜位置为硬编码，没有随布置类型适配

**目标**：根据台数 N 和电机功率自动选择布置形式，计算各主要空间坐标，并更新 SVG 渲染。

---

## 推荐方案：规则驱动 + 布置模板选择

按工程规范内置判断树，根据 N 选择布置模板，在模板内套用现有尺寸公式。

**决策规则**（依据《排水泵排布规则》第6节）：

| 台数 N | 布置形式 |
|--------|---------|
| N ≤ 4 | 单排（保持现状） |
| N > 4 | 双排对置（新增） |

**单排**（现有，不变）：
```
L = N × w_pump + (N-1) × d_spacing + 2 × e_wall
W = max(2.5, d_pump + aisle + wall)
```

**双排对置**（新增）：
```
n1 = ceil(N/2)    // 第一排台数（近集水池侧）
n2 = floor(N/2)   // 第二排台数（近出水侧）
L  = n1 × w_pump + (n1-1) × d_spacing + 2 × e_wall  （向上取0.1m）
W  = 2 × d_pump + 1.2 + 2 × 0.3                      （aisle=1.2m，wall_clr=0.3m，向上取0.1m）
```

## 实施步骤

### Step 1：runAG12 扩展（index.html ~477行）

```javascript
// 在现有 const w_pump/d_pump 之后
const layoutType = N > 4 ? 'double' : 'single';

if (layoutType === 'single') {
  // 现有逻辑不变
  return { L, W, d_spacing, e_wall, w_pump, d_pump, layoutType };
} else {
  const n1 = Math.ceil(N / 2), n2 = Math.floor(N / 2);
  const L_raw = n1 * w_pump + (n1 - 1) * d_spacing + 2 * e_wall;
  const W_raw = 2 * d_pump + 1.2 + 2 * 0.3;
  const L = ceilTo01(L_raw), W = Math.max(2.5, ceilTo01(W_raw));
  return { L, W, d_spacing, e_wall, w_pump, d_pump, layoutType, n1, n2 };
}
```

### Step 2：renderAG12 扩展（index.html ~591行）

在结果卡片中新增一行：
- 布置形式：单排 / 双排对置
- 双排时显示"第一排 n1 台 / 第二排 n2 台"

### Step 3：runAG31 平面图扩展（index.html ~780行）

提取泵坐标计算为局部函数，分支处理两种布置：

```javascript
// 单排（现有逻辑封装）
// 双排对置：
//   第一排 n1 台：py = room_y2 - (0.3 + d_pump) * ps （靠集水池侧）
//   第二排 n2 台：py = room_oy + 0.3 * ps             （靠出水侧）
//   进水管：每台泵均独立接到集水池（R-LA-04）
//   出水管：两排各自汇入出水总管
//   控制柜：维护间右端短边角落（单排/双排统一）
```

### Step 4：A-A 剖面图

W 已在 Step 1 按双排公式更新，剖面图代码以 W 为宽度，比例尺 `ss` 自动适配，**无需修改剖面图逻辑**。

## 验证方案

1. **N=2, P<55kW**：布置=单排，L/W 与现有结果完全一致（回归）
2. **N=4, P<55kW**：布置=单排，4台泵一字排开
3. **N=5, P<55kW**：布置=双排，n1=3/n2=2，L 明显短于单排N=5，SVG 中双排可见
4. **N=6, P>55kW**：双排 + 大间距（d_spacing=1.2m），L/W 均正确加大
5. 剖面图水位线、池深标注在所有场景下仍正确
6. 缩放/平移交互在双排图形下正常工作
