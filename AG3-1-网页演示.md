# AG3-1 网页演示 Agent

## 一、职责概述

AG3-1 是雨水泵房智能设计系统的**最终集成 Agent**，负责将 AG0-0、AG1-1、AG1-2、AG2-1 的全部逻辑转换为 JavaScript 代码，并以单一 HTML 网页（`index.html`）的形式提供完整的交互式功能演示。

用户在网页中输入三个核心参数（Q、N、S），点击计算后，页面依次调用各 Agent 的 JS 实现，展示每一步的计算过程和最终结果。

---

## 二、在系统中的位置

```
用户（浏览器）
  │  输入：Q, N, S
  ▼
AG3-1（index.html / JavaScript）
  ├── 调用 AG0-0：输入校验
  ├── 调用 AG1-1：单泵规格计算  → Q_single, D_nominal
  ├── 调用 AG1-2：维护间尺寸计算 → L, W
  └── 调用 AG2-1：水池体积深度计算 → V_min, h_pool, H
  │
  ▼
结果展示（分步计算过程 + 最终结论）
```

## 三、输入参数

| 参数名 | 符号 | 类型 | 单位 | 说明 | 约束 |
|--------|------|------|------|------|------|
| 总设计流量 | Q | Float | m³/h | 泵站总排水量 | Q > 0 |
| 工作泵台数 | N | Int | 台 | 不含备用泵 | 1 ≤ N ≤ 6 |
| 集水池占地面积 | S | Float | m² | 集水池平面面积 | S > 0 |
| 电机功率（可选） | P | Float | kW | 影响泵间距选取 | 默认 30 kW |
| 出水口超高（可选） | h_out | Float | m | 影响扬程计算 | 默认 1.0 m |
| 出水管总长（可选） | L_pipe | Float | m | 影响沿程损失 | 默认 50 m |

function runAG11(Q, N) {
  const Q_single = Q / N;
  const q = Q_single / 3600;                         // m³/s

### 5.4 AG1-2：维护间尺寸计算

```js
function runAG12(N, motorPower=30, w_pump=0.6, d_pump=0.8) {
  const d_spacing = motorPower > 55 ? 1.2 : 1.0;   // R-LA-01/03
  const e_wall    = motorPower > 55 ? 1.0 : 0.8;   // R-LA-02/03
  const L = ceilTo1(N*w_pump + (N-1)*d_spacing + 2*e_wall);
  const W = Math.max(2.5, ceilTo1(Math.max(1.2, d_pump+0.5) + 0.3));
  return { d_spacing, e_wall, w_pump, d_pump, L, W, steps };
}
```

### 5.5 AG2-1：水池体积深度计算

```js
function runAG21(Q_single, N, S, h_out=1.0, L_pipe=50) {
  const Z       = 6;
  const V_min   = Q_single / (4 * Z);               // R-CM-01
  const h_active = Math.max(0.5, V_min / S);
  const h_pool  = ceilTo1(h_active + 1.1);          // +0.5淹没+0.3沉淀+0.3超高
  const stopLevel  = 0.5;
  const startLevel = stopLevel + h_active;
  const alarmLevel = startLevel + 0.3;
  // 扬程 R-HY-02
  const H_static   = h_pool + h_out;
  const H_friction = L_pipe * 0.003;
  const H_local    = H_friction * 0.4;
  const H = (H_static + H_friction + H_local) * 1.08;
  // 梯级启泵水位 R-CL-01
  const startLevels = Array.from({length:N}, (_,i) => startLevel + i*0.2);
  return { Z, V_min, h_active, h_pool,
           stopLevel, startLevel, alarmLevel, startLevels,
           H_static, H_friction, H_local, H, steps };
}
```
