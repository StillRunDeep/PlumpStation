# AG1-1 单泵规格计算 Agent

## 一、职责概述

AG1-1 负责根据泵站总设计流量 Q 和工作水泵台数 N，计算**单台水泵的设计流量**，并据此确定**排水管（出水管）的公称内径**，为水泵选型和管路设计提供基础参数。

---

## 二、在系统中的位置

```
AG0-0（用户对话）
  │  输入：Q（总流量）, N（台数）
  ▼
AG1-1（单泵规格计算）
  │  输出：Q_single（单泵流量）, D_nominal（排水管公称内径）
  ▼
后续水泵选型 / 管路设计
```

---

## 三、输入参数

| 参数名 | 类型 | 单位 | 来源 | 说明 |
|--------|------|------|------|------|
| Q | Float | m³/h | AG0-0 | 泵站总设计流量 |
| N | Int | 台 | AG0-0 | 工作水泵台数（不含备用泵） |

---

## 四、输出参数

| 参数名 | 类型 | 单位 | 说明 |
|--------|------|------|------|
| Q_single | Float | m³/h | 单台水泵设计流量，对应 `Pump.flowRate` |
| v_pipe | Float | m/s | 排水管设计流速（取值范围内） |
| D_inner | Float | mm | 排水管计算内径 |
| D_nominal | Int | mm | 排水管公称内径（按标准管径向上取整） |

---

## 五、计算逻辑

### 步骤 1：计算单泵设计流量

```
Q_single = Q / N
```

- Q_single 对应本体对象 `Pump.flowRate`
- 各台水泵取相同流量（依据规则 **R-OP-01** 推荐同型号水泵）

**示例：** Q = 360 m³/h，N = 3，则 Q_single = 120 m³/h

---

### 步骤 2：确定排水管设计流速

排水管（出水管）流速 v 依据 **R-HY-02（系统总扬程计算规则）** 及管路水力设计要求：

| 管径范围 | 推荐流速 v |
|----------|-----------|
| DN ≤ 150 mm | 0.8 ～ 1.0 m/s |
| DN 200 ～ 300 mm | 1.0 ～ 1.2 m/s |
| DN ≥ 350 mm | 1.2 ～ 1.5 m/s |

> 默认取 v = 1.0 m/s 作为初始估算值，后续可根据实际管径迭代修正。

---

### 步骤 3：计算排水管计算内径

将单泵流量换算为 m³/s：

```
q = Q_single / 3600        （m³/s）
```

根据连续性方程，管道过流面积：

```
A_pipe = q / v             （m²）
```

排水管计算内径：

```
D_inner = sqrt(4 × q / (π × v))    （m）
D_inner_mm = D_inner × 1000        （mm）
```

---

### 步骤 4：向上取标准公称内径

依据给水排水标准管径系列，取不小于 D_inner_mm 的最近标准值：

| 标准公称内径 DN（mm） |
|-----------------------|
| 100, 125, 150, 200, 250, 300, 350, 400, 450, 500, 600, 700, 800 |

选取原则：**D_nominal ≥ D_inner_mm**，取系列中满足条件的最小值。

**示例：**
- Q_single = 120 m³/h，v = 1.0 m/s
- q = 120/3600 = 0.0333 m³/s
- D_inner = sqrt(4 × 0.0333 / (π × 1.0)) = 0.206 m = 206 mm
- 取标准值 D_nominal = **DN250**

---

### 步骤 5：验证工作点效率

确认所选水泵在计算工作点下，运行效率满足规则 **R-HY-03（水泵工作点效率规则）**：

```
η_operating ≥ 0.85 × η_BEP
```

- η_BEP：水泵最高效率点（BEP）效率，来自 `PumpCharacteristicCurve.bestEfficiencyPoint`
- 若实际工作点偏离高效区，应调整 N 或更换水泵型号

---

## 六、输出汇总示例

| 输入 | 值 |
|------|----|
| Q（总流量） | 360 m³/h |
| N（台数） | 3 |

| 输出 | 值 |
|------|----|
| Q_single（单泵流量） | 120 m³/h |
| v_pipe（设计流速） | 1.0 m/s |
| D_inner（计算内径） | 206 mm |
| D_nominal（公称内径） | DN250 |

---

## 七、引用的本体对象与规则

**对象（Object）：**
- `Pump`：flowRate（单泵流量）、efficiency（效率）
- `OutletPipe`：diameter（出水管管径）
- `PumpCharacteristicCurve`：bestEfficiencyPoint（最高效率工作点）

**规则（Rule）：**
- **R-HY-02** 系统总扬程计算规则：管路流速范围 0.8～1.5 m/s
- **R-HY-03** 水泵工作点效率规则：η_operating ≥ 0.85 × η_BEP
- **R-OP-01** 同型号水泵规则：建议各台水泵采用相同型号（Q_single 相同）
- **R-OP-02** 并联泵台数限制规则：工作泵台数 N ≤ 6
