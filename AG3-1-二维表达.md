# AG3-1 二维表达 Agent

## 一、职责概述

AG3-1 负责将前序 Agent 的计算结果转化为**二维泵房平面示意图**，在网页画布（Canvas）上精准绘制泵房内的主要设备对象，所有设备的尺寸与位置均与计算结果保持一致。

目标：让设计人员在获得数值结果的同时，直观看到泵房平面布局。

---

## 二、在系统中的位置

```
AG1-1 → Q_single, D_nominal
AG1-2 → L（维护间净长）, W（维护间净宽）, d_spacing, e_wall, w_pump, d_pump
AG2-1 → h_pool, V_min, stopLevel, startLevel, alarmLevel
AG0-0 → N（水泵台数）
  │
  ▼
AG3-1（二维表达）
  │  输出：Canvas 平面图（SVG/HTML5 Canvas）
  ▼
网页展示区域
```

| 参数名 | 来源 | 单位 | 说明 |
|--------|------|------|------|
| N | AG0-0 | 台 | 工作水泵台数 |
| L | AG1-2 | m | 维护间净长（水泵排列方向） |
| W | AG1-2 | m | 维护间净宽（通道方向） |

| e_wall | AG1-2 | m | 端部水泵距侧墙净距 |
| w_pump | AG1-2 | m | 单台水泵外形宽度 |

| S | AG0-0 | m² | 集水池占地面积（推算集水池平面尺寸） |

## 四、绘制内容与规则

### 4.1 平面图坐标系

以维护间左下角为原点，X 轴向右（水泵排列方向），Y 轴向上（通道方向）。

### 4.2 绘制对象清单

| 对象 | 本体类 | 绘制方式 | 定位依据 |
|------|--------|----------|----------|
| 维护间外轮廓 | RainfallPumpStation | 矩形边框，L × W | 全局边界 |
| 集水池 | CollectionPool | 矩形，面积 = S，宽度紧贴维护间一侧 | 与维护间共享一侧墙 |
| 水泵机组 × N | Pump + Motor | 矩形块，w_pump × d_pump | 一字排开，间距 d_spacing，距墙 e_wall |
| 进水管 × N | InletPipe | 连接水泵与集水池的短线段 | 每台泵独立取水（R-LA-04） |
| 出水管 × N | OutletPipe | 从水泵延伸至汇总干管的线段 | 汇集到维护间对侧墙 |
| 出水总管 | OutletPipe（干管） | 横向干管，贯通全部水泵出口 | 紧靠维护间背墙 |
| 止回阀 × N | CheckValve | 出水支管上的符号（菱形） | 每台泵出水管上 |
| 闸阀 × N | GateValve | 出水支管上的符号（矩形） | 止回阀外侧 |
| 控制柜 | ControlCabinet | 矩形块 | 维护间端部角落（R-LA-08） |
| 尺寸标注 | — | 双向箭头 + 文字 | L、W、d_spacing、e_wall、w_pump |

### 4.3 集水池平面推算

集水池占地面积 S 已知，平面宽度（与维护间共享方向）取等于维护间净长 L，深度（垂直方向）为：

```
D_pool_plan = S / L    （m）
```

集水池绘制在维护间的进水侧（Y 轴负方向），两者共享一道隔墙。

### 4.4 缩放比例

画布尺寸固定为 760 × 500 px（可随容器自适应），自动计算比例尺：

```
scale = min(canvas_width / (L + D_pool_plan + margin),
            canvas_height / (W + margin))   （px/m）
```

## 五、图例与标注

- **设备图例**：右上角彩色方块 + 名称
- **尺寸标注**：蓝色细线 + 箭头，标注 L、W、d_spacing、e_wall
- **比例尺**：左下角横条，显示实际长度（如「── 1 m」）
- **设备标签**：每台水泵编号（P1、P2…），集水池标注 h_pool 和 V_min

---

## 六、引用的本体对象与规则

**对象（Object）：**
- `RainfallPumpStation`：顶层容器，确定外轮廓
- `Pump`：机组矩形块，w_pump × d_pump
- `CollectionPool`：集水池矩形，面积 = S
- `InletPipe`：独立进水管线段（R-LA-04）
- `OutletPipe`：出水支管 + 干管
- `CheckValve`：出水管止回阀符号
- `GateValve`：出水管闸阀符号
- `ControlCabinet`：就地控制柜矩形

**规则（Rule）：**
- **R-LA-01** 水泵间距规则：d_spacing ≥ 1.0/1.2 m → 绘图间距
- **R-LA-02** 泵与墙壁间距规则：e_wall ≥ 0.8/1.0 m → 绘图边距
- **R-LA-04** 进水管独立取水规则：每台泵独立进水管
- **R-LA-08** 就地控制面板高度规则：控制柜布置在角落
- **R-OP-05** 布置对称性规则：水泵左右对称排布

## 七、输出

Canvas 元素嵌入网页 AG3-1 结果卡片，与其他 Agent 结果并列显示。支持导出为 PNG（可选）。
